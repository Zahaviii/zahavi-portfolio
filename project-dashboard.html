Dashboard CMS

April 25, 2026
CodeIgniter Bootstrap MySQL HeidiSQL PHP

Dashboard CMS is a web-based content management system built using CodeIgniter, Bootstrap, and MySQL via HeidiSQL. The system is designed to give administrators full control over the content displayed on the main landing page, making it easy to manage and update without touching the code.

The system features a secure login and registration system, ensuring that only authorized administrators can access the dashboard. Once logged in, admins can manage all content from a single, clean, and intuitive dashboard interface that is fully responsive across all devices.

One of the core features is the content management system where admins can input text and images that will be displayed directly on the main landing page. The video management feature allows admins to add video thumbnails and links that will appear on the homepage, making it easy to showcase video content without any technical knowledge.

The system also includes a user management feature where admins can monitor and view all registered users in the system. Additionally, a news management feature allows admins to create and manage news articles complete with titles, content, and thumbnails, keeping the website content fresh and up to date.

To complete the system, a shopping cart and checkout feature has been integrated, allowing users to browse, add items to their cart, and complete purchases seamlessly. All these features are combined into one powerful and easy-to-use system, making Dashboard CMS a complete solution for managing dynamic web content.

Features:

  • Login & Registration system
  • Admin-only dashboard access
  • Content management (text & images for landing page)
  • Video management (thumbnail & video link)
  • User management (view registered users)
  • News management (title, content & thumbnail)
  • Shopping cart & checkout system
